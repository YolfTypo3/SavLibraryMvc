============
Users Manual
============

SAV Library Mvc is designed to work with extensions generated
by the `SAV Library Kickstarter <https://extensions.typo3.org/extension/sav_library_kickstarter>`_.

#. Download the extension `SAV Library Mvc 
   <https://extensions.typo3.org/extension/sav_library_mvc>`_ and install it.

#. If you want to test the library, you can install the test
   extension `SAV Library Mvc Example0 <https://extensions.typo3.org/extension/sav_librarymvc_example0>`_ 
   available in the TER.
  
#. If you want to generate extensions, download the extension `SAV Library Kickstarter 
   <https://extensions.typo3.org/extension/sav_library_kickstarter>`_ and install it.
   Read the :ref:`SAV Library Kickstarter tutorial section <savlibrarykickstarter:tutorial>`.
